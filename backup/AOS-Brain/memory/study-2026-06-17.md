## Logic Systems

### Propositional Logic
- **AND (∧)**: Both true → true
- **OR (∨)**: Either true → true  
- **NOT (¬)**: Inverts truth value
- **XOR (⊕)**: Exclusive OR, one true but not both
- **IMPLIES (→)**: If A then B

### Truth Tables
| A | B | A∧B | A∨B | A⊕B | A→B |

### Predicate Logic
- **∀ (FOR ALL)**: ∀x P(x) - true for all x
- **∃ (THERE EXISTS)**: ∃x P(x) - true for at least one x

### Fuzzy Logic
