# Milk Man Game
## The Dairy Avenger of Dairyopolis

**Platform:** DroidScript (Android)  
**Genre:** Side-scrolling platformer / shooter  
**Theme:** Dairy / Comedy / Action

---

## 🎮 GAME CONCEPT

**Premise:**  
You are Milk Man, the dairy-powered defender of Dairyopolis. The villainous **Vil Laine** and **Madame Shoezete** threaten the city's milk supply.

**Tagline:** *"Got milk? Got justice!"*
